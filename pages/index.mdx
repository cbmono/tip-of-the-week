# Tip of the Week

Every week a team member of Digital Product & Tech will present something this person learned that could be useful to peers. Topics can be new tools or software, useful CLI commands, shortcuts, etc.

![ALTEOS](/tip-of-the-week.png)

## Rules

<Steps>

### 25 Minutes meeting

- 10 minutes presentation.
- 10 minutes for questions and discussion.
- 5 minutes for setup and time buffer.

### The Tip should be presented in a short and concise way

- What is the Tip?
- Why is it useful?
- How to use it?
- Example of use and where to find more information.

### The presentations need to be added into `./presentations`

- The file should be named `YYMMDD-<your-topic>.mdx`.
- Use [000000-template.mdx](/presentations/000000-template.mdx) as a template.
- In case you need to add images or other files, create a folder with the same name as the file in `./public` and add the files there.

</Steps>

---

<Callout type="warning" emoji="⚠️">
  Please do not upload any sensitive information in your presentations. In the
  future, this might be a public repository.
</Callout>
